# Swiss Civic Lab

Prototype GitHub Pages pour présenter des projets de démocratie numérique en Suisse:
vote électronique, e-collecting, chatbot civique, idées citoyennes et inclusion numérique.

## Structure

- `index.html` - page principale
- `styles.css` - identité visuelle et responsive design
- `script.js` - onglets interactifs
- `assets/hero-civic-tech.png` - image de couverture générée pour le projet

## Publication GitHub Pages

Dans GitHub:

1. Ouvrir `Settings`.
2. Aller dans `Pages`.
3. Choisir `Deploy from a branch`.
4. Sélectionner la branche `main` et le dossier `/root`.

Le site sera ensuite disponible à l'adresse GitHub Pages du dépôt.
